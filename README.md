# Easy Clock - 悬浮时钟

一个简单的桌面悬浮时钟应用，基于 Electron 构建。

## 功能特性

- 悬浮在桌面上的圆形模拟时钟显示
- 实时更新时间和日期
- 平滑移动的时钟指针
- 完整的数字刻度标记（1-12）
- 可拖拽移动位置
- 右键菜单提供更多选项
- 双击隐藏/显示时钟
- 关闭按钮退出应用

## 安装与运行

1. 克隆或下载此项目
2. 进入项目目录
3. 安装依赖：
   ```
   npm install
   ```
4. 启动应用：
   ```
   npm start
   ```

## 使用说明

- **移动时钟**：点击并拖拽时钟窗口即可移动位置
- **隐藏/显示**：双击时钟区域可以隐藏或显示时钟
- **右键菜单**：
  - 始终置顶：切换时钟是否始终置顶显示
  - 透明度：调整时钟背景透明度（高/中/低）
  - 退出：关闭时钟应用
- **关闭应用**：点击右上角的 × 按钮

## 时钟特色

- 圆形模拟时钟界面，带有小时、分钟和秒钟指针
- 平滑移动的时针、分针和秒针效果，无晃动现象
- 时钟面上有12个刻度标记，标有完整的1-12数字，使用数学计算精确定位
- 实时更新的数字时间显示（包含秒数）
- 现代化设计，带有渐变和阴影效果
- 完整显示整个时钟界面，包括上半部分和下半部分

## 自定义配置

你可以在 [config.js](config.js) 文件中修改默认配置：

- 窗口大小和样式
- 时间和日期格式
- 背景颜色和透明度等

## 开发

```bash
# 开发模式运行
npm run dev

# 构建应用
npm run build
```

## 打包成Windows可执行文件

```bash
# 打包Windows应用
npm run dist
```

打包完成后，可执行文件将位于 `dist/win-unpacked` 目录中，文件名为 `Easy Clock.exe`。

你也可以使用以下命令创建一个简化版本的可执行文件：

```bash
# 创建简化版可执行文件
npm run package
```

这将创建一个 `EasyClock.exe` 文件在项目根目录中。
